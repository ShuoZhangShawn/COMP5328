## How to run
### Installation

<code>pip install -r requirements.txt</code>

### Usage
1. copy the dataset file under **/data** folder
2. unzip the dataset zip file
3. the structure of data folder should be 

<code>
data/
└── 2024_A2_datasets/
    ├── CIFAR10.npz
    ├── FashionMNIST0.3.npz
    └── FashionMNIST0.6.npz
</code>

Run /Assignment2_final.ipynb
